# conduit-services

This is starter code for node-js applications.

## Installation

###Build Docker image

#### dev
```sh
$ ./dockerTask.sh build debug
```
#### release
```sh
$ ./dockerTask.sh build release
```

###Spun up docker container

#### dev
```sh
$ ./dockerTask.sh compose debug
```

#### release
```sh
$ ./dockerTask.sh compose release
```
